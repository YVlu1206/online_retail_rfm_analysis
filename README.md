# online_retail_rfm_analysis
​基于英国零售数据的用户行为探索性分析 (EDA) 与 RFM 客户分层模型，旨在识别核心价值用户并提供针对性的运营策略建议。
# 英国零售电商用户行为与 RFM 分层分析

## 项目简介
本项目基于 100 万行英国电商原始数据，利用 Python 进行端到端的数据处理与分析。

## 核心工作流
1. **数据清洗**：剔除 25% 的匿名用户数据，识别并处理退货订单及异常大额订单。
2. **行为探索 (EDA)**：
   - 发现用户下单峰值集中在 12:00-15:00（午间碎片化购物习惯）。
   - 确认销售额受订单量驱动而非单一客单价驱动。
3. **RFM 建模**：
   - 将 5881 名用户划分为核心、重要、普通及流失四个维度。

## 业务洞察与建议
- **二八法则验证**：核心用户（约 8%）贡献了主要营收，建议建立 VIP 会员权益。
- **营销策略**：针对午间活跃高峰，建议在此时间段推送运营通知。
